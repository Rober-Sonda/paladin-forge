## Post técnico (LinkedIn)
Hoy publiqué AgentForge, una herramienta de consola para coordinar flujos multi-agente de extremo a extremo.

¿Qué resuelve?
- Visibilidad total del pipeline (pending/running/done/failed)
- Delegación explícita por agente/subagente/modelo
- Estimación de tokens y costo por sesión/modelo/agente
- Gestión de catálogo para personalizar el equipo
- Ayuda guiada y edición directa de archivos de configuración desde terminal

Pensado para equipos que quieren velocidad sin perder control operativo.

🔗 Repo: https://github.com/Rober-Sonda/paladin-forge
