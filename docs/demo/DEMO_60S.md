# AgentForge Demo 60s

## Objetivo
Mostrar en 1 minuto: visibilidad total, delegación por agentes, costos estimados y edición en consola.

## Guion (tiempo real)
- 00s-08s: abrir consola `agentforge-console --project /tmp/agentforge-demo`
- 08s-15s: opción `18` para activar `Duplicate screen mode`
- 15s-24s: opción `13` para enseñar `Help center`
- 24s-35s: opción `14` para enseñar `Token & cost dashboard`
- 35s-46s: opción `16` para mostrar creación/edición de agentes
- 46s-55s: opción `17` para abrir editor en consola
- 55s-60s: opción `7` (dry-run) para ver handoff e indicadores en vivo

## Comandos sugeridos
1. `agentforge init /tmp/agentforge-demo --template full-delivery --force`
2. `agentforge-console --project /tmp/agentforge-demo`
