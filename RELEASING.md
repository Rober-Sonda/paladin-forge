# Releasing AgentForge

## First public release checklist
1. Review `README.md` and screenshots.
2. Run `./install.sh` locally.
3. Validate:
   - `agentforge list-agents`
   - `agentforge-console --project .`
4. Commit all files.
5. Tag release: `v0.1.0`.
6. Publish GitHub release notes from `CHANGELOG.md`.

## Suggested tags
- `v0.1.0` initial public release
- `v0.2.0` keyboard navigation and themes
- `v0.3.0` export sessions and project templates
